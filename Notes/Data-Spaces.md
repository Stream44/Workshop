- https://github.com/livegnik/2way-poc-design - 2WAY is a protocol and backend for software that stays correct without a central authority. It gives apps a local-first foundation for identity, permissions, ordering, storage, and sync, enforced by the system. Each device, user, and app keeps its own keys and history and enforces the same rules. Changes are accepted only if valid, authorized, and structurally sound.
